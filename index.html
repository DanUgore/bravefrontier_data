Just a page to fool around with the Brave Frontier data.
Open up the console and use the BFData object.
BFData.files contains the data by filename / path. For example BFData.files.info leads to info.json and BFData.files.jp.info leads to jp/info.json
BFData.Units contains the data in info.json
BFData.Items contains the data in items.json
BFData.Evolutions contains the data in evo_list.json
and BFData.JP contains the same properties but for the JPBF data.
